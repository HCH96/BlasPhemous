# BlasPhemous

C++과 WinAPI 기반 자체 엔진으로 제작한 2D 액션 플랫포머 모작 프로젝트입니다.  
원작 Blasphemous의 전투, 이동, 보스 패턴, 스테이지 연출을 참고하여 플레이어 액션, 몬스터 AI, 보스전, 충돌, 애니메이션, 카메라 시스템을 직접 구현했습니다.

## Features

- 플레이어 이동, 점프, 회피, 패링, 연속 공격, 주문 공격 구현
- FSM 기반 플레이어 / 몬스터 / 보스 상태 관리
- 스프라이트 메타 데이터 기반 애니메이션 로딩
- 카메라 추적, 화면 제한, 흔들림, 페이드 연출 구현
- 패럴랙스 배경 렌더링
- 회전 콜라이더와 플랫폼 충돌 처리
- 보스 패턴, 투사체, 스포너 시스템 구현
- 체크포인트, 리스폰, UI, 사운드 시스템 구현
